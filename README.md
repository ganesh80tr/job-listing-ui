# Job Listing UI

A responsive Job Listing UI built with React and Vite.

## Features

* Browse job listings
* Responsive design
* Clean and modern UI
* Fast development with Vite

## Tech Stack

* React
* Vite
* JavaScript
* CSS

## Installation

```bash
git clone https://github.com/ganesh80tr/job-listing-ui.git
cd job-listing-ui
npm install
npm run dev
```

## Build for Production

```bash
npm run build
```
